# Valfre Instagram Auto 🚀

Painel automático para gerar conteúdo de Instagram puxando produtos do site.

## Funcionalidades
- Busca produtos automaticamente do site
- Gera copy persuasiva com IA
- Salva histórico de posts
- Interface visual via Streamlit

## Deploy

1. Suba os arquivos no GitHub
2. Vá para https://share.streamlit.io
3. Conecte o repositório
4. Escolha app.py
5. Deploy

## Secrets necessários

No Streamlit → Settings → Secrets:

OPENAI_API_KEY="SUA_CHAVE"
SITE_URL="https://www.ferramentasvalfre.com.br"
